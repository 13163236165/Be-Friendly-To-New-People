## SVM(支持向量机)源代码


### 依赖环境

python3.7

### python库

numpy 

matplotlib

### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd SVM支持向量机
pip install -r requirements.txt
```

### 使用方法

在当前目录下输入

```
cd SVM支持向量机
```

运行PCA算法

```
python SVM.py
```
