## 微信读书答题辅助python小程序


### 依赖环境

python3.7

### python库

* urllib
* Pillow
* BeautifulSoup4
* base64
* requests
* time
* re


### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd 微信读书答题辅助
pip install -r requirements.txt
```

### 使用方法

在当前目录下输入

```
cd 微信读书答题辅助
```

运行程序

```
python question-answer.py
```
