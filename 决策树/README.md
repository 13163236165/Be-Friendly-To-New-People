## 决策树代码


### 依赖环境

python3.7

### python库

numpy 

matplotlib

### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd PCA
pip install -r requirements.txt
```
### 文件介绍
* datasets 为数据文件
* decision-tree.py 为决策树主函数文件
* treePlotter.py 使用matplotlib绘画决策树

### 使用方法

在当前目录下输入

```
cd 决策树
```

运行PCA算法

```
python decision-tree.py
```
