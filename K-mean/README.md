## K-mean、二分K-mean聚类算法
主要是解决无标签数据分类问题，文件的名字就是对应的算法

### 依赖环境

python3.7

### python库

numpy 

matplotlib

### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
pip install -r requirements.txt
```

### 使用方法

在当前目录下输入

```
cd K-mean
```

运行K-mean算法

```
python K-mean.py
```

运行二分K-mean算法

```
python 二分K-mean.py
```
