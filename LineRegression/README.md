## 线性回归算法源代码


### 依赖环境

python3.7

### python库

numpy 

matplotlib

### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd LineRegression
pip install -r requirements.txt
```

### 使用方法

在当前目录下输入

```
cd LineRegression
```

运行线性回归算法

```
python LineRegression.py
```
