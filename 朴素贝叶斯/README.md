## 朴素贝叶斯源代码


### 依赖环境

python3.7

### python库

numpy 


### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd 决策树
pip install -r requirements.txt
```
### 文件介绍
* datasets 为数据文件 里面的ham表示非垃圾邮件 spam 表示垃圾邮件
* 朴素贝叶斯.py 为朴素贝叶斯主要算法


### 使用方法

在当前目录下输入

```
cd 朴素贝叶斯
```

运行朴素贝叶斯算法

```
python 朴素贝叶斯.py
```
