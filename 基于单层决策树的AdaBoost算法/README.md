## 基于单层决策树的AdaBoost算法


### 依赖环境

python3.7

### python库

numpy 

matplotlib

### 下载安装

```
git clone https://github.com/lzx1019056432/Be-Friendly-To-New-People.git
```

### 安装依赖包

```
cd 基于单层决策树的AdaBoost算法
pip install -r requirements.txt
```
### 文件介绍
* datasets 为数据文件
* 基于单层决策树的AdaBoost.py 为决策树主函数文件


### 使用方法

在当前目录下输入

```
cd 基于单层决策树的AdaBoost算法
```

运行AdaBoost算法

```
python 基于单层决策树的AdaBoost.py
```
